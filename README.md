# Account Ruiner
Rapes accounts.

# Requirements
- Node.js (https://nodejs.org)

# Instructions
- Open a command prompt in this folder
- Run `npm install`
- Open the start.bat
- Replace `TOKEN_HERE` with the token you want to destroy
- Rape the account
